# Unnecessary Enchantable Brush

![Mod Logo](mod_title.png)

### Overview

**Unnecessary Enchantable Brush** is a mod that makes Minecraft's Brush item enchantable, along with 2 exclusive
enchantments for Brush:

- **Swift Hand**: _TBA_
- **Lucky Finds**: Each level adds 25% chance to obtain additional item count from brushing. (up to level 4)
